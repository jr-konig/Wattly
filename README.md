# Wattly

Real-time PC power usage and electricity cost tracker with historical analytics.


Features
- Estimates PC power consumption
- Tracks electricity cost over time
- Daily / weekly / monthly / yearly breakdown
- Local-first data storage (no cloud)

---

How it works:
Voltly estimates system power usage based on runtime metrics and converts it into cost using local electricity pricing.
